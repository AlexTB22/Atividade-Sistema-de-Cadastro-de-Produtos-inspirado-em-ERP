# Atividade-Sistema-de-Cadastro-de-Produtos-inspirado-em-ERP
o programa está no a2.py
programa em linguagem Python que simula o módulo de estoque de um ERP, permitindo ao usuário cadastrar, excluir, listar e visualizar relatórios de produtos.

primeiramente precisa instalar as bibliotecas matplotlib e tabulate com os seguintes códigos no terminal:
pip install matplotlib
pip install tabulate

usei sqlite3 para criar e gerenciar o banco de dados
usei tabulate para visualizar os registros
